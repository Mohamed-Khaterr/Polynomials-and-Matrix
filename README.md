# Polynomials & Matrix
This App to calculate Root of Polynomials and Matrix
1. The Root of Polynomials with five different methods:
    1. Bisection
    2. False Position
    3. Simple Fixed Point
    4. Netwon
    5. Secant
2. Matrix with three different methods:
    1. Gauss Elimination
    2. LU Decomposition
    3. Cramer's Rule
