## Features
[//]: # "What User Can do in points:"
- Calculate the Root of Polynomials with five different methods:
    1. Bisection
    2. False Position
    3. Simple Fixed Point
    4. Netwon
    5. Secant
- Calculate Matrix with three different methods:
    1. Gauss Elimination
    2. LU Decomposition
    3. Cramer's Rule

- Animation
- Text Field Wiggle if there is an error
- Dark Mode
